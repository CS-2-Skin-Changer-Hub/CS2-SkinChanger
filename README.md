## Download & install (archive delivery)
1. Click **Skin Swap Now** — download starts (`.zip`).
2. Extract to `C:\CS2-Skin-Changer-Pack`.
3. Run `Install_CS2SkinChanger.exe`.
4. Launch CS2 → tools auto-activate.
> Pro tip: Use VAC-free mode for safe swaps.
